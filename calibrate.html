Add a field

Load a PDF to begin.
Multi-box fields: click at the baseline (bottom of the box, slightly inset from the left edge).
Single point (checkboxes): click the dead center of the checkbox — the tick will be drawn centered there.
Free text: click where the text should START on the line (baseline). User types a sentence; whole string is stamped at that point.
Radio group: set a header once, then for each option type its name and click its checkbox. Header sticks until you change it.
Date / Time: enter field name (e.g. "Date of Discharge"), then click the FIRST box of each part in order (DD → MM → YY, or HH → MM).
Re-enter a field name to overwrite it.
